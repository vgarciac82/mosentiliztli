# Evidencia de cumplimiento

Esta carpeta es el **destino único** para la evidencia de cumplimiento de checklists de seguridad y auditoría (p. ej. checklist obligatorio IDOR de la Dirección General de Ciberseguridad).

Incluye: reportes, diagramas de autorización, resultados de pruebas de seguridad (IDOR, negativas, manipulación de parámetros), evidencias de monitoreo y registros de eventos (extractos anonimizados, configuración de alertas, procedimientos de rotación).

- **Convención de nombres (IDOR):** `IDOR-evidencia-<sistema>-<tipo>-<fecha>.md` o `IDOR-evidencia-<identificador-auditoria-o-ticket>.md`
- **Referencia:** [ai-specs/specs/security-standards.md](../ai-specs/specs/security-standards.md) — Parte V (Cumplimiento checklist IDOR y evidencia)
